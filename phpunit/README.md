# Instructions

## Installation

* Install php,
  * `brew tap homebrew/dupes`
  * `brew tap josegonzalez/homebrew-php`
  * `brew install php54`
* Install composer (package manager)
  * `brew install josegonzalez/php/composer`
* Install dependencies `composer install`
* Run tests with `./run-test.sh`
